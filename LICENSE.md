## 🐛 Common Issues & Fixes

**❌ Script doesn't execute / nothing happens**
> Ensure your executor is injected *after* Roblox has fully loaded. Wait for the main game screen before injecting.

**❌ GUI doesn't appear on mobile**
> Make sure you're using a supported mobile executor (Arceus X Neo or Hydrogen). Older versions of Arceus X are not compatible.

**❌ Kill Aura not hitting enemies**
> Open the GUI settings and increase the Kill Aura radius. Some NPCs require a radius above 20 to be detected reliably.

**❌ Auto Dungeon gets stuck**
> This usually happens after a Roblox update changes dungeon geometry. Check the Changelog for a recent patch, or temporarily disable path smoothing in the settings tab.

**❌ Script kicks me on load**
> You may be using an outdated executor. Update to the latest version, or switch to a different supported executor listed in the Compatibility table.

**❌ "attempt to index nil" error in console**
> The game hasn't fully loaded before the script ran. Add a 5-second delay before executing, or re-inject after the character spawns.

**❌ Auto Farm stops after a few minutes**
> Enable the **Anti-AFK** toggle in the utility tab to prevent Roblox from disconnecting idle sessions.
